Техническое задание: Веб-приложение "Мафия" с видеозвонком
1. Общее описание
Веб-приложение для проведения игры "Мафия" с поддержкой видеозвонков и автоматизацией игровой механики. Интерфейс разделен на две основные зоны: верхняя - видеозвонок (Zoom integration), нижняя - игровая панель управления.
2. Функциональные требования
2.1. Основные модули
Модуль 1: Видеозвонок (верхняя часть экрана)
•	Интеграция с Zoom API
o	Возможность создания/присоединения к конференции
o	Отображение участников в grid-формате
o	Управление микрофоном/камерой
o	Кнопка "Поднять руку" для выступления
Модуль 2: Игровая механика (нижняя часть экрана)
2.2.1. Подготовка к игре
•	Создание комнаты
o	Уникальный ID комнаты
o	Настройка списка ролей
o	Установка таймеров
•	Присоединение игроков
o	Ввод никнейма
o	Выбор аватарки
o	Подтверждение готовности
2.2.2. Распределение ролей
•	Список ролей (8 позиций):
1.	Мирный житель
2.	Мафия
3.	Босс Мафии
4.	Комиссар
5.	Адвокат
6.	Доктор
7.	Маньяк
8.	Зеркало
•	Автоматическое распределение или ручное назначение ведущим
•	Приватные уведомления о роли каждому игроку
2.2.3. Игровой процесс
•	Система фаз:
o	Ночь (последовательность пробуждения)
o	Утро (объявление результатов)
o	День (обсуждение)
o	Голосование
•	Ночная фаза:
text
Порядок действий:
1. Зеркало → выбор цели
2. Адвокат → выбор защиты
3. Комиссар → проверка игрока
4. Мафия → выбор жертвы
5. Маньяк → выбор жертвы
6. Доктор → выбор лечения
o	Каждой роли показывается свой интерфейс выбора
o	Таймер на принятие решения (30-60 секунд)
o	Подтверждение выбора
•	Дневная фаза:
o	Отображение результатов ночи
o	Таймер обсуждения (2-5 минут)
o	Система поднятия руки для очереди выступлений
o	Чат для текстовых сообщений
•	Голосование:
o	Каждый игрок выбирает кандидата
o	Отображение результатов в реальном времени
o	Автоматический подсчет голосов
o	Обработка способности Адвоката
2.2.4. Специфические механики
•	Зеркало:
o	Выбор цели при активации
o	Логика перенаправления выстрела мафии
o	Визуализация эффекта
•	Адвокат:
o	Выбор защиты на ночь
o	Автоматическая отмена казни при совпадении
o	Уведомление о срабатывании
2.3. Интерфейс ведущего
•	Панель управления:
o	Кнопки управления фазами
o	Принудительное завершение хода
o	Редактирование таймеров
o	Управление списком игроков
•	Информационная панель:
o	Таблица ролей и состояний
o	История ночных действий
o	Статистика голосований
o	Секретные данные (кто кого проверял/лечил)
•	Модерация:
o	Исключение игроков
o	Пауза игры
o	Сброс голосования
